# Side-Channel Analysis of AES using Transformer, LSTM, YOLO and Hybrid Deep Learning Models

This repository contains the implementation accompanying my M.Sc. thesis at Khalifa University.

The project investigates deep learning architectures for profiling-based side-channel analysis (SCA) of AES using the ASCAD benchmark.

The evaluated architectures include:

- LSTM
- Transformer (TransNet-inspired)
- YOLOv8
- YOLO11
- YOLO–Transformer Hybrid
- HDT-Gated YOLO–Transformer Hybrid

Experiments were conducted on:

- ASCAD Fixed
- ASCAD_desync50
- ASCAD_desync100

The ASCAD dataset is publicly available from ANSSI.

Download it from:

https://github.com/ANSSI-FR/ASCAD

Update the dataset paths inside the notebooks before running the experiments.
